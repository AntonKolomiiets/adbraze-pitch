A cleaner, faster way to start the day

Picking a task feels like a one-click setup

When a designer hits Start work, everything that used to take several minutes kicks off automatically.

The extension automatically:

  • Switches the task to Motion In-Process in AdBraze
  • Creates a new local folder with the task name
  • Creates a fresh AE project inside that folder
  • Parses the description and finds referenced assets
  • Downloads those assets into the folder
  • Imports them into AE, renaming where needed
  • Creates a paired Dropbox folder for outputs
  • Links that Dropbox folder back into the AdBraze task metadata

The result: the task is ready to animate in seconds ‚Äî no manual detective work, no searching for assets, no ‚Äúwhere is the .aep?‚Äù chats in Telegram.

Rendering & delivery that just works

When the designer hits Complete inside the extension, the delivery flow is handled end-to-end.

The extension:

  • Sets up the AE Render Queue to render to the correct Dropbox folder
  • Uploads final renders directly into the linked Dropbox
  • Auto-attaches those renders to the AdBraze task
  • Moves the task to Motion Review
  • Shows a mandatory ‚ÄúCollect Project‚Äù step (no skipping)
  • Requires the designer to collect files so versioning and dependencies stay intact
  • Prevents broken tasks and missing assets further down the line

Only after collection is done does the extension unlock the ability to start the next task.

Why this matters for the company:

  • No more missing project files
  • No broken paths or links
  • No ‚ÄúI‚Äôll collect tomorrow‚Äù delays
  • Everything stays consistent across the team
Rejected tasks are clearly visible inside the extension and can be picked up and fixed immediately, keeping the feedback loop tight.
